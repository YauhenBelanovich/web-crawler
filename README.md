## Getting started

### _for the project start it is required:_

  * Git
  * Docker
  * Docker-compose

### _commands:_
```
$ git clone -b develop https://github.com/YauhenBelanovich/web-crawler.git
$ cd web-crawler
$ docker-compose up -d
```
